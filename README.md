## 🛠️ Installation & Setup

### 📥 Install Dependencies
```bash
npm install
```

### 🏃‍♂️ Run Development Server
```bash

```

### 🧪 Run Tests
```bash
npm test
```

### 📦 Build for Production
```bash
npm run build
```
